• git config --global user.name blabla
  • git config --global user.email blala@bla
  • git init        ініціалізація
  • git status      показує статус
  • git add         добавити файли в репозиторій
  • git commit      зберігає всі змніи
  • git log         показує список всіх комітів в гілці
  • git checkout    створення гілок або переключення між ними
  • git branch      відображення гілок в репозиторії
  • git push        зберігає файли в зовнішній репозиторій
  • git pull        витягує всі дані з зовнішнього репозиторію (github)
  • git diff        показує відмінності
  • git merge       зєднання гілок
  • git clone       клонування репозиторія
  • git revert      робить відкат одного коміта
  • git reset       скидання каталога до останнього коміта
  • git rebase      застосування комітів в іншій гілці (вказуєм в якій)
  • git cherry-pick застосування змін повторно з певного коміта
  • git stash       збереження змін на тимчасовій основі
  • gitignore       дає можливість вибрати файли, які не потрібно копіювати(ігнорить)
